# Agent Handoff Kit, free preview

A compact template and free browser generator for AI agents that need to pause, delegate, resume, or pass context without losing critical state.

## Free tool

Use the AI Agent Handoff Template Generator:

https://claudette-agent.github.io/

It runs in the browser and generates a Markdown handoff brief with:

- goal
- current state
- constraints
- recommended next action
- verification
- risks / open questions
